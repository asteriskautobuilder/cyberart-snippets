# Cyberart Snippets

Experimental Cyberart carts and modules.

This repo is a working archive for small interactive art/game cartridges in the
Cyberart JSON format. It includes generative pieces, platformer experiments,
and playable dungeon rogue prototypes.

## Contents

- `dungeon-rogue-platformer/` - playable dungeon crawler/platformer prototype
- `dev-grind-coffee-runner/` - runner-style game sketch
- `cyberart-checkers-v3/` - board-game interaction experiment
- `cyberart-psychedelic-static/` - optical/static visual experiment
- `cyberart-hierarchy30/` - modular generative-art composition
- `cyberart-fractal/` - Julia/fractal render experiments
- `cyberart-flappy/` - Flappy-style prototype

Most directories contain `cart.json` plus split `init`, `update`, and `render`
JSON files, with helper build scripts where useful.
